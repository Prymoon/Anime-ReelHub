## ✨ Features

- 🎮 Dedicated sections for Anime and Games content
- 🎨 Beautiful animations powered by Anime.js
- 🌟 User-friendly interface with smooth transitions
- 📱 Fully responsive design
- 🎵 Background music with toggle control
- 👤 User session management
- 🔒 Secure admin panel
- 📤 Easy video upload system
- 🖼️ Custom thumbnail support
- 🎯 Category-based organization

## 🚀 Quick Start

1. Clone this repository to your XAMPP htdocs folder:
```bash
git clone https://github.com/Priyanshu84iya/Anime-ReelHub.git
```

2. Create the database:
- Import create_database.sql to your MySQL server
- Default database name: reel_db

3. Access the application:
```
http://localhost/Reel
```
